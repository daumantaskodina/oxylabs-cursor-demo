# Cursor Workshop (Mac)

You are a product designer. You have Cursor. Let's go.

## Step 1: Run the project

Open [cursor-computer-setup.md](docs/workshop/cursor-computer-setup.md), paste the prompt into Cursor, and it will set everything up and start the app.

## Step 2: Edit with Figma

Open [figma-mcp-setup.md](docs/workshop/figma-mcp-setup.md), add Figma to Cursor, paste the prompt with your Figma link, and Cursor will build it into the app.
