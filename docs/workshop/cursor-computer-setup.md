# Cursor Computer Setup

Paste this prompt into Cursor chat.

```text
I am a product designer running this project for the first time on a Mac.
I may not have all the required tools installed yet.

Please do the following:
1) Check if Homebrew, Node.js, npm, Git, and GitHub CLI are installed.
2) Install anything missing using Homebrew.
3) Install project dependencies.
4) Start the app and tell me the URL to open in my browser.
```

That's it. Cursor will handle the rest.

Two commands to remember once it is running:

- Restart app: `npm run dev`
- Stop app: `Ctrl + C`
